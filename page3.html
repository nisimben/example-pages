CSS Positioning – Page 3: Layers & z-index

In this exercise, you'll create a layered composition using position: relative on the container and position: absolute on the children. Adjust the z-index values to control which element appears on top. Try experimenting with removing or changing the z-index values to see how the stacking order changes.

Poster Example

Task: Understand the back, middle and top layers of this poster. The background rectangle, title text and badge are stacked using z-index. Change their values in the CSS to reorder the layers.

Practice Ideas

  • Move the badge to a different corner by adjusting top, right, bottom or left values.
  • Create another badge and give it a smaller z-index so it appears below the first badge.
  • Nudge the title by adding a small position: relative shift, e.g. left: 10px.
  • Use opacity on the background layer to make it semi-transparent.
